# WookiePedia

A simple app that fetches data from https://swapi.dev/

<img src="WookiePedia.gif" width=200 />

## Installation

Just run Yarn install

```bash
yarn install
```

## Usage

```bash
npx react-native run-ios
OR
npx react-native run-android
```
